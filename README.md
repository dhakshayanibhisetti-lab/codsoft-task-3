# Restaurant Menu Mobile UI Design

This project is a modern and clean Restaurant Menu Mobile UI designed in Figma for the CODSOFT UI/UX Internship.

## Project Overview
The aim of this project is to create a visually appealing and user-friendly restaurant menu application interface that helps users browse food items easily.

## Features
- Modern mobile UI design
- Search bar for food items
- Food categories section
- Beautiful food cards
- Ratings and delivery time
- Food details screen
- Add to Cart button
- Bottom navigation bar

## Screens Included
1. Home/Menu Screen
2. Food Details Screen

## Tools Used
